# pesto-pdf-omni-crawler-merger-protocol
Pesto PDF crawler and merger protocol

To ensure the functional deployment of Pesto PDF 3.2, the environment must include the FPDF Library and the FPDI extension suite. Developers are required to download and install the following dependencies from their respective official repositories:

FPDF Library
Available at: https://www.fpdf.org/

FPDI v2.6.5 & FPDI PDF-Parser
Available at: https://www.setasign.com/news-archive/fpdi-2.6.5-and-fpdi-pdf-parser-2.1.5-released/

Once the open-source packages have been retrieved, move both library directories into the project root directory. After these assets are correctly mapped within the file system, the core engine will be fully operational. Happy coding to the entire community.
